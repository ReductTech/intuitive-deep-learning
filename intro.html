3.4 中心极限定理

从条件、定理到证明，按教材顺序阅读。

传统教材
第三章 · 概率分布的极限定理

定理 3.4.1　设随机变量序列 X1, X2, … 相互独立且同分布，满足 E(Xi) = μ，Var(Xi) = σ² < ∞。 令 Sn = Σni=1Xi，则当 n → ∞ 时，有

(Sn − nμ) / (σ√n) ⟶d N(0, 1) 即　limn→∞ P {(Sn − nμ) / (σ√n) ≤ x} = Φ(x)

推论 3.4.1　记样本均值 X̄n = Sn / n，则 √n(X̄n − μ) / σ 依分布收敛于标准正态随机变量。因而当 n 足够大时，可用正态分布近似样本均值的分布。

证明　由特征函数在原点邻域的展开，结合独立性可得标准化和的特征函数逐点收敛于 exp(−t²/2)。根据特征函数的唯一性与连续性定理，标准化随机变量依分布收敛于 N(0, 1)，故结论成立。□

注：上述结论要求随机变量独立同分布且具有有限方差；一般情形需另行验证相应条件。

Meta skill

创作 skill 的 skill

Meta 直译是“元”，更接地气的说法是“套娃”：meta review 是对 review 再 review，meta evaluation 是评价评价指标。Meta skill，就是创作 skill 的 skill。

Skill 也不只是“技能”，而是一个由提示词、各种脚本和素材组成、可以被智能体调用的小单元。

Meta skill=设计哲学+适配框架
核心
目标

智能体可在有限人工干预下，完成内容初步生成。

01 / Design philosophy

设计哲学

○

无意识设计

尽量使操作方式与所学概念一致，减少额外的界面学习成本。理想情况下，学生通过简短提示即可理解下一步操作。

理解距离，就在数轴上靠近目标；理解分类边界，就直接在两类样本之间画线。
↘

渐进式概念披露

控制信息和概念出现的顺序，每个阶段尽量只增加一个主要变量，在具体问题出现后再引入相应术语和工具。

先呈现问题和现象，再逐步补充术语、公式与实现细节。
≈

直觉式学习

在有限学习时间内，优先帮助学生理解方法所解决的问题、适用条件和主要限制，并形成能够迁移到其他任务的基本判断。

例如，与记忆排序算法的逐行实现相比，更关注学生能否根据数据和任务约束选择合适的方法。
↗

个性化

根据学习者的基础、目标与操作轨迹，调整内容顺序、提示强度与练习难度。

让同一概念拥有不同入口，学习路径随人而变。
