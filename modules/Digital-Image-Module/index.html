数字图像如何变成 RGB 矩阵

先观察一张图，再把图片拆成计算机能读的三个数字表。

请先用手机摄像头靠近这张图

打开手机摄像头，尽量放大并对准图像的白色和彩色区域。观察后，用一句话记录你看到的细节。

每个像素都由红、绿、蓝三种强度表示

三个数字写成 RGB：红色强度 R、绿色强度 G、蓝色强度 B。纯红色是 [1, 0, 0]。

你的任务 把像素调成纯绿色 RGB = [0, 1, 0]：R 调到 0，G 调到 1，B 调到 0。
0–1 强度 [1.00, 0.00, 0.00]
8 bit 存储 [255, 0, 0]

0–1 表示颜色强度；8 bit 把每个通道存为 0–255，所以 1 对应 255，0 对应 0。

尚未完成：请让 RGB 变成 [0, 1, 0]。

现在换成你自己的图片

选择一张图片，看看它怎样按 RGB 约定拆成三个通道。图片只在本地处理，不会上传。
