为什么需要损失函数

从距离误差开始，观察损失函数怎样衡量预测和目标之间的差距。

损失就是距离

Loss 衡量真实值和预测值之间差多少。先用一条数轴，把这种差距直接画出来。

你的任务 拖动绿色预测值，让它与红色真实值重合，把 Loss 缩小到 0。
L1 Loss = |真实值 - 预测值| 5.4
L2 Loss = (真实值 - 预测值)² 29.2
操作提醒： 拖动绿色预测值，让它贴近红色真实值。

你的任务 先算出当前预测的 L1、L2 损失。计算正确后，再解释两种损失的区别。

计算 L1 与 L2 的梯度

损失只告诉我们错了多少；梯度进一步告诉模型，预测值应该往哪个方向改，以及这一步需要多强的修正。

怎样读梯度 梯度的正负表示修正方向，绝对值表示当前误差产生的修正强度。训练时会沿梯度的反方向更新预测。

L1 Loss

L₁(ŷ, y) = |ŷ - y|
∂L₁/∂ŷ = sign(ŷ-y)

L2 Loss

L₂(ŷ, y) = (ŷ-y)²
∂L₂/∂ŷ = 2(ŷ-y)
